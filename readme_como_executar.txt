Trabalho Prático implementação de um programador para a Linguagem P 
Amaury Ribeiro

Como executar:

Abra o terminal e navegue até o diretório com a pasta contendo os arquivos de código, e execute o comando:
	python compilador_p.py 

Também é possivel realizar apenas a análise léxica executando o arquivo lexico.py
	python lexico.py
	
Após a execução será perguntado o número correspondente ao caso de teste que deseja utilizar (por padrão os casos de teste são lidos nas pastas do diretório input)

Observação:
O código foi escrito e testado com a versão do Python 3.9.5
